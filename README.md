Please keep in mind that these scripts have only been tested in python3.

- antiSMASH v5.0
- CORASON
- BiG-MAP
- dREP
- GTDB-Tk
- Pyhton 3+

#### Pyhton Packages:

- argparse
- pandas
- numpy
- collections
- json
- SeqIO
- subprocess
